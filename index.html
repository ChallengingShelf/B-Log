Dissertation / Honours Project

Investigation into the use of Ethereum for Internet of Things (IoT) node transaction authentication through smart contracts.

Utilised MetaMask, blockchain technologies and Cooja network simulation tools to produce a proof-of-concept research paper exploring enhanced security within IoT environments.

TLDR:

Using blockchain for IoT node transaction authentication is incredibly effective however it is complete overkill. The low powered and lightweight nature of IoT devices is not suitable for the proposed resource-heavy authentication methodology. However, in the future, with advances in computing power and battery technology this method could become viable.
